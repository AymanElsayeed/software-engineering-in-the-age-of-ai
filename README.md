# Software Engineering in the Age of AI

This repository contains tasks, use cases, and experiments related to the course  
**"Software Engineering in the Age of AI"**, as part of my MSE studies.

---

## 🎯 Purpose

This repository explores how AI tools (especially LLMs) impact Software Engineering.

The main goal is to understand:
- How to effectively use AI tools in development workflows
- When to rely on AI — and when not to
- The limitations and challenges AI introduces into engineering practices

---

## 🧠 Topics Covered

- How to write effective prompts
- Prompt engineering methods and techniques
- Practical AI use cases in Software Engineering
- Experimentation with AI-assisted development
